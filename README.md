Kaynak Kısıtlı IoT Sistemlerinde Veri Bütünlüğü Simülasyonu
Bu çalışma, donanım kaynakları sınırlı olan IoT (Nesnelerin İnterneti) cihazlarında veri bütünlüğünün korunması amacıyla geliştirilen güvenlik protokollerinin performans analizine odaklanmaktadır.
Metodoloji: 
Çalışma kapsamında, 100 byte'lık ham veri (raw data) belirlenen kriptografik algoritmalar ile Python ortamında şifrelenerek 'hesaplama gecikmesi' ve 'paket boyutu genişlemesi' (overhead) metrikleri teorik olarak 
elde edilecektir. Elde edilen bu veriler, OMNeT++ simülasyon ortamına parametrik girdi olarak aktarılacaktır. Simülasyonlarda; farklı ağ mimarilerinin (Wi-Fi, Zigbee, LoRaWAN vb.) bu ek güvenlik yükü altındaki 
uçtan uca gecikme (end-to-end latency) performansları, maksimum iletim biriminin (MTU) aşılmasına bağlı paket parçalanmaları ve bu artışın sebep olduğu darboğazların modüler etkileri incelenecektir.
